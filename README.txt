Steps to run file on Windows.
Install MCR if needed. Can be found at https://www.dropbox.com/sh/2t1dpjzfk552u4z/AADVxREqrNiY6WHN-PWSXVlna?dl=0. Add to PATH variable. example of default C:\Program Files\MATLAB\MATLAB Compiler Runtime\v715\runtime\win64
1. Open command prompt
2. Go to folder where the SDAD_CS.exe file in downloaded
3. Type SDAD_CS Group1.csv Group2.csv delta alpha Highpr depth
example SDAD_CS Doctorate.csv Bachelors.csv 0.1 0.05 0.95 3
4. The results will be stored in a SDAD_contrasts.csv file in the same folder.
